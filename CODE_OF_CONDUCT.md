# Contributor Covenant 行为准则

## 我们的承诺

我们作为成员、贡献者和领导者，承诺让每个人都能在社区中免于骚扰，不论年龄、体型、残障、族裔、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、种姓、肤色、宗教或性取向。

我们承诺以有助于开放、友好、多元、包容、健康社区的方式行事和互动。

## 我们的标准

有助于营造积极环境的行为包括：

- 对他人表现出同理与善意
- 尊重不同的观点、经验和想法
- 得体地给予和接受建设性反馈
- 承担责任并向受影响者道歉，从经历中学习
- 关注对社区整体最有利的事情，而不仅限于个人

不可接受的行为包括：

- 使用与性有关的语言或图像，以及任何形式的性关注或搭讪
- 挑衅、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人私人信息（如地址、邮箱）
- 其他在专业环境中可合理视为不当的行为

## 执行责任

社区领导者有责任澄清和执行我们的行为标准，并对任何其认为不当、威胁、冒犯或有害的行为采取适当且公平的纠正措施。

社区领导者有权并有责任删除、编辑或拒绝与行为准则不符的评论、提交、代码、Wiki 编辑、Issue 等贡献，并在适当时说明 moderation 决定。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共空间中正式代表社区的情况。代表社区的例子包括使用官方邮箱、通过官方社媒账号发帖、或作为指定代表参与线上或线下活动。

## 执行

若遭遇滥用、骚扰或其他不可接受的行为，可向负责执行的维护者报告（通过 [GitHub Issues](https://github.com/namelesserlx/tiptap-codeless/issues) 联系项目维护者）。所有投诉将得到及时、公正的审查与处理。

社区领导者有义务尊重报告者隐私与安全。
