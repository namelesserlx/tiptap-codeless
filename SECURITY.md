# 安全政策

## 支持的版本

我们会对以下版本发布安全更新：

| 版本       | 支持        |
| ---------- | ----------- |
| 最新主版本 | ✅ 支持     |
| 旧版本     | ❌ 不再支持 |

## 报告漏洞

如发现安全漏洞，请**不要**开公开 Issue。

请通过 [GitHub Security Advisories](https://github.com/namelesserlx/tiptap-codeless/security/advisories/new) 提交**私密安全报告**，或通过仓库维护者联系方式私下说明问题。我们会尽快确认并回复。

感谢你负责任地披露问题。
