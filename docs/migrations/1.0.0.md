# 1.0.0 Migration Guide / 迁移指引

## Overview / 概述

This release rewrites the public API surface for all three extensions. Every old
option name is replaced with a canonical 1.0 equivalent. There are **no
compatibility aliases** — you must update your configuration to match the new
names.

> 此版本重写了三个扩展的公共 API。所有旧的选项名称都被替换为规范的 1.0 版本，**不保留兼容别名** —— 必须更新配置。

## code-block-pro

### Before → After

```ts
// 0.x
CodeBlockPro.configure({
  macosControls: { close: true, collapse: true, fullscreen: true },
  toolbar: {
    showLanguageSelector: true,
    showCopyButton: true,
    showLineNumbersToggle: true,
  },
  lineNumbers: {
    startLine: 1,
    toggleable: true,
  },
  collapse: {
    collapsedLines: 5,
  },
  lazyRender: {
    enabled: true,
    rootMargin: '100px',
  },
  ui: {
    languageDropdown: { zIndex: 1000 },
  },
  className: 'my-code-block',
})

// 1.0.0
CodeBlockPro.configure({
  windowControls: { close: true, collapse: true, fullscreen: true },
  toolbar: {
    language: true,
    copy: true,
    lineNumbers: true,
  },
  lineNumbers: {
    enabled: true,
    start: 1,
    allowToggle: true,
  },
  collapse: {
    enabled: true,
    defaultCollapsed: false,
    visibleLines: 5,
  },
  rendering: {
    lazy: true,
    rootMargin: '100px',
    placeholderHeight: 100,
  },
  ui: {
    layers: { languageDropdown: 1000 },
  },
  HTMLAttributes: { class: 'my-code-block' },
})
```

### Option map / 选项对照

| 0.x | 1.0.0 |
|-----|-------|
| `macosControls` | `windowControls` |
| `toolbar.showLanguageSelector` | `toolbar.language` |
| `toolbar.showCopyButton` | `toolbar.copy` |
| `toolbar.showLineNumbersToggle` | `toolbar.lineNumbers` |
| `lineNumbers.startLine` | `lineNumbers.start` |
| `lineNumbers.toggleable` | `lineNumbers.allowToggle` |
| `collapse.collapsedLines` | `collapse.visibleLines` |
| `lazyRender` | `rendering` |
| `ui.languageDropdown.zIndex` | `ui.layers.languageDropdown` |
| `className` | `HTMLAttributes.class` |

### New / 新增

```ts
CodeBlockPro.configure({
  locale: 'en',                              // zh-CN / zh-TW / en / ja
  messages: {
    controls: { closeCodeBlock: 'Close' },   // override specific labels
  },
})
```

## drag-handle

### Before → After

```ts
// 0.x
DragHandle.configure({
  element: {
    insert: <MyIcon />,
    drag: <MyDragIcon />,
  },
  handleStyle: {
    width: 32,
    height: 32,
    zIndex: 100,
  },
  drag: {
    dragOpacity: 0.5,
  },
  insertMenu: {
    itemsMode: 'replace',
    position: 'left',
    offset: { x: 10, y: 0 },
  },
  ui: {
    menu: { component: MyMenu },
  },
  triggerOnSlash: true,
  triggerOnInput: false,
  includeOnlyNodes: ['heading', 'paragraph'],
  excludeNodes: ['image'],
})

// 1.0.0
DragHandle.configure({
  handle: {
    width: 32,
    height: 32,
    zIndex: 100,
    iconSize: 20,
    icons: {
      insert: <MyIcon />,
      drag: <MyDragIcon />,
    },
    hoverDelay: 0,
    hideDelay: 100,
  },
  drag: {
    enabled: true,
    opacity: 0.5,
  },
  insertMenu: {
    enabled: true,
    strategy: 'replace',
    placement: 'left',            // 'right' | 'left' | 'top' | 'bottom'
    offset: { x: 10, y: 0 },
    trigger: '/',                 // string | RegExp | false
    zIndex: 1000,
    items: [...],
    component: MyMenu,
  },
  nodes: {
    include: ['heading', 'paragraph'],
    exclude: ['image'],
  },
  events: {
    onDragStart: (info, event) => {},
    onDragEnd: (info, event) => {},
    onNodeChange: (info) => {},
    onInsertClick: (info, event) => {},
  },
})
```

### Option map / 选项对照

| 0.x | 1.0.0 |
|-----|-------|
| `element.insert` | `handle.icons.insert` |
| `element.drag` | `handle.icons.drag` |
| `handleStyle` | `handle` (consolidated) |
| `drag.dragOpacity` | `drag.opacity` |
| `insertMenu.itemsMode` | `insertMenu.strategy` |
| `insertMenu.position` | `insertMenu.placement` + `insertMenu.offset` |
| `ui.menu` | `insertMenu.component` |
| `triggerOnSlash` / `triggerOnInput` | `insertMenu.trigger` |
| `includeOnlyNodes` | `nodes.include` |
| `excludeNodes` | `nodes.exclude` |

### New / 新增

```ts
DragHandle.configure({
  locale: 'en',
  messages: {
    tooltip: { dragToMove: 'Drag to move' },
  },
  offset: { x: -32, y: 0 },   // global handle offset
})
```

## file-upload

### Before → After

```ts
// 0.x
FileUpload.configure({
  storageMode: 'local',
  localStorageOptions: {
    directoryHandle: myHandle,
    alwaysAskDirectory: true,
  },
  upload: async (files, ctx) => { ... },
  accept: 'image/*,video/*',
  multiple: true,
  handlePaste: true,
  handleDrop: true,
  allowedMimeTypes: ['image/png', 'image/jpeg'],
  maxFileSize: 10 * 1024 * 1024,
  imgBubbleMenuConfig: {
    enabled: true,
    zIndex: 9999,
  },
})

// 1.0.0
FileUpload.configure({
  storage: {
    mode: 'local',                    // 'memory' | 'base64' | 'local' | 'custom'
    directoryHandle: myHandle,
    alwaysAskDirectory: true,
    upload: async (files, ctx) => {   // only required when mode === 'custom'
      return { assets: [...] };
    },
  },
  picker: {
    accept: 'image/*,video/*',
    multiple: true,
  },
  ingest: {
    paste: true,
    drop: true,
    allowedMimeTypes: ['image/png', 'image/jpeg'],
    maxFileSize: 10 * 1024 * 1024,
  },
  ui: {
    bubbleMenu: {
      enabled: true,
      zIndex: 9999,
    },
    uploadPlaceholder: {
      enabled: true,
    },
  },
})
```

### Option map / 选项对照

| 0.x | 1.0.0 |
|-----|-------|
| `storageMode` | `storage.mode` |
| `localStorageOptions.directoryHandle` | `storage.directoryHandle` |
| `localStorageOptions.alwaysAskDirectory` | `storage.alwaysAskDirectory` |
| `upload` | `storage.upload` |
| `accept` | `picker.accept` |
| `multiple` | `picker.multiple` |
| `handlePaste` | `ingest.paste` |
| `handleDrop` | `ingest.drop` |
| `allowedMimeTypes` | `ingest.allowedMimeTypes` |
| `maxFileSize` | `ingest.maxFileSize` |
| `imgBubbleMenuConfig` | `ui.bubbleMenu` |

### New / 新增

```ts
FileUpload.configure({
  locale: 'en',
  messages: {
    fileCard: { downloadFile: 'Download' },
  },
  ui: {
    uploadPlaceholder: { enabled: true },   // show progress placeholder during upload
  },
  onError: (err) => console.error(err),
})
```

## Steps to Migrate / 迁移步骤

1. Replace each extension's `.configure({...})` call with the new option names
   above.
2. Remove any workarounds that depended on the old option names.
3. If using local file storage (`storage.mode: 'local'`), treat 1.0.0 metadata
   as the new baseline for asset rehydration.
4. Run your editor integration tests against the new builds.
5. If any option is not listed above, it was kept unchanged.

> 1. 按上方的对照表替换每个扩展的 `.configure({...})` 调用。
> 2. 移除依赖旧选项名称的补偿代码或 workaround。
> 3. 如果使用本地文件存储 (`storage.mode: 'local'`)，将 1.0.0 的元数据格式作为新的基准。
> 4. 用新版本重新运行编辑器集成测试。
> 5. 未在上方列出的选项保持不变。
